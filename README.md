## 프로젝트 소개

최근 취미 생활을 즐기는 ‘갓생러’ 들이 증가함에 따라, <br>
이들이 자유롭게 활동할 수 있는 공간을 제공하는 것을 목적으로, <br>
같은 취미를 가진 사람들이 모여 교류하고 소통할 수 있는 커뮤니티를 구축하여 <br>
초보자들의 진입장벽을 낮추고, 부담 없이 새로운 취미에 도전할 수 있도록 지원합니다
